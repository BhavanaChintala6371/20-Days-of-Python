## Overview

This repository contains the Day 1 Python revision material for the 1-week Python learning plan. It covers the foundation required before moving to later topics: Python introduction and setup, syntax and output, comments, variables, data types, numbers, casting, and strings.

The complete study material is available in [`notes.md`](notes.md).

## Recommended Study Flow

1. Read [`notes.md`](notes.md) section by section.
2. Type every important example yourself instead of only reading it.
3. Create `day1_revision.py` using the master revision code in the notes.
4. Run the file and check its output line by line.
5. Change values and observe how the output changes.
6. Build the four mini projects in separate `.py` files.
7. Create `day1_full_revision_practice.py` and solve at least 25 practice problems.
8. Complete the final checklist before moving to Day 2.

## Mini Projects

1. **Personal Profile Report** - clean and format a name, city, goal, and email.
2. **Lead Reply Report** - calculate and display reply and meeting percentages.
3. **Company Name Cleaner** - clean a company name and inspect its string details.
4. **Email Checker** - normalize an email and check its domain and ending.

## Completion Checklist

You are ready to move ahead when you can:

- Create and run a `.py` file.
- Use `print()` and comments correctly.
- Create clear variables and valid variable names.
- Work with `str`, `int`, `float`, `bool`, and `None`.
- Use `type()`, `int()`, `float()`, and `str()`.
- Perform basic arithmetic.
- Index, slice, check, clean, join, and format strings.
- Use f-strings, `:.2f`, and escape characters.
- Solve at least 25 Day 1 practice problems without copying the answers.

## Main Learning Rule

Do not only read the code. Type it, run it, check the output, modify the values, and observe what changes.
